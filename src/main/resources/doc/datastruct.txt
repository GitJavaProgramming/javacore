存储结构
    数据结构在计算机中的表示称为数据的存储结构
    顺序存储 元素逻辑和物理上都相邻 物理相邻依赖底层C语言支持 java.lang.reflect.Array类提供了动态创建和访问Java数组的静态方法。
    链式存储 元素逻辑上相邻
线性表：数据元素之间仅有线性关系，每个数据元素只有一个直接前驱和一个直接后继。(实数的定义？)
    这样一种容器有三种增加和删除数据的方式：
        栈：先进后出
        队列：先进先出
        链表：在结构前后添加，任意位置删除，需要遍历
        散列表：通过散列函数找到存储位置
树：数据元素之间有着明显的层次关系，并且每一层的数据元素和下一层的多个元素相关，但是只能和上一层的一个元素相关。（点线？）
    class Node<E> {
        E elem;
        Node<E> parent;
        List<Node<E>> children;
    }
    树、二叉树、完全平衡二叉树、红黑树、B树、B+树、B*树、线段树...
图：由顶点和顶点的边构成的离散结构，其中数据元素之间的关系是任意的。（面/体？）

研究Java数据结构的一般方法
    1、构造结点--元数据结构 确定存储结构
    2、定义方法--算法 算法依赖数据结构去实现 数据结构限定了在其自身上作用的算法
    3、增加功能--软件扩展
        迭代  定义位置和结点的对应关系
        导航  定位插入元素的最近位置
        阻塞队列
        并发
        ...
